**高级非托管虚拟机磁盘：每个帐户的限制**

| 资源 | 默认限制 |
| --- | --- |
| 每个帐户的总磁盘容量 |35 TB |
| 每个帐户的总快照容量 |10 TB |
| 每个帐户的最大带宽（传入 + 传出<sup>1</sup>） |<=50 Gbps |
