# SQL 数据仓库数据管理合作伙伴
若要创建端到端数据仓库解决方案，请从行业领先的各种工具中进行选择。本文重点介绍其数据管理工具和解决方案支持 Azure SQL 数据仓库的 Microsoft 合作伙伴公司。
